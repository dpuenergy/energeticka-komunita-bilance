Balíček a CLI `ecb` pro výpočet energetické bilance a vyhodnocení sdílení v energetických komunitách (EK).

## Rychlý start
```bash
python -m venv .venv && source .venv/bin/activate
pip install -e .[dev]
ecb --help
```
